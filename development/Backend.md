# Setup up env
Install venv :
> foo@app/back$> python3 -m venv env
> foo@app/back$> source env/bin/activate
> foo@app/back$> pip install -r env/requirements.txt

Activate virtual env :
> foo@app/back$> source env/bin/activate

Desactivate virtual env : 
> foo@app/back$> deactivate
